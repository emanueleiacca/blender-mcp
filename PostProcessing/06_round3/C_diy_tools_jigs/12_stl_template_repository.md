## 12.9 Workshop organizer

### Search term

- https://www.printables.com/search/models?q=gridfinity

### Sistema Gridfinity (raccomandato!)

**Gridfinity** = sistema modulare open source per workshop organization
inventato da Zack Freedman (https://www.youtube.com/c/ZackFreedman).
- Grid base 42 × 42 mm modulare stampato.
- Vaschette, scatole, holder custom per ogni tool.
- Migliaia di modelli compatibili community-driven.

Search: https://gridfinity.xyz / https://www.printables.com/search/models?q=gridfinity

Costo per workshop 1 m² scaffale: **~500g PLA = €12** vs scaffalatura
Wera/Sortimo industriale €200-400.

## 12.11 Licenze e fair use

- **CC-BY**: attribuzione obbligatoria, uso commerciale OK.
- **CC-BY-NC**: attribuzione + **NO uso commerciale** (vendita pezzi
  stampati = violazione; uso interno workshop = OK).
- **CC-BY-SA**: share-alike (chi modifica deve rilasciare con stessa
  licenza).
- **CC0**: dominio pubblico, tutto permesso.
- **Closed/Proprietary**: leggi note autore.

**Per piccolo brand commerciale R2**: scarica STL per **uso interno
workshop** (jig, organizer, holder) = sempre OK indipendentemente da
licenza. NON rivendere mai gli STL stessi. Pezzi finali del brand
(vasi, sculture, ecc.) devono essere modelli **tuoi** o licenze
commerciali esplicite.

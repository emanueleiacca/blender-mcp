# 06 — Sourcing EU + Brexit + Dogana: guida operativa Italia

Per import da **USA, UK, JP, CN** verso Italia 2025-2026. Linee guida tasse, tempi, costi nascosti.

> Verificare sempre normativa aggiornata su **agenziadoganemonopoli.gov.it** e **europa.eu/taxation_customs**.

---

## A. Regimi IVA UE (riepilogo 2025-2026)

### A.1 Cosa è cambiato dal 1° luglio 2021

- **Abolita** la franchigia IVA per beni importati da extra-UE sotto **€22**. Tutti i beni pagano IVA italiana **22%** (per la maggior parte degli articoli hobby/consumer).
- **IOSS (Import One Stop Shop)**: regime opzionale per venditori extra-UE per riscuotere IVA al checkout su ordini ≤ **€150**. Se il venditore è IOSS, il pacco entra senza dogana fisica e senza handling fee del corriere → **costo finale ridotto**.
- **Soglia €150**: sotto, solo IVA. Sopra, IVA + dazi doganali + handling.

### A.2 Cosa significa per l'utente IT che compra hobby/spray/filamenti

| Origine | Valore ordine | Cosa pago |
|---------|---------------|-----------|
| **Italia / EU** (DE, FR, ES, PL, CZ, NL, ecc.) | qualsiasi | Solo prezzo + spedizione, IVA già inclusa nel prezzo (mercato unico) |
| **UK** (post-Brexit) | ≤ €150 | IVA 22% + handling corriere €5-25, **se** venditore non ha IOSS o equivalente UK-EU |
| **UK** | > €150 | IVA 22% + dazio (variabile per categoria, spesso 0% per filamenti/strumenti) + handling €15-35 |
| **USA** | ≤ €150 | IVA 22% + handling €5-15, **se** venditore non IOSS |
| **USA** | > €150 | IVA 22% + dazio + handling €15-50+ |
| **JP** (Mr. Hobby diretto) | ≤ €150 | IVA 22% + handling, raramente venditori JP usano IOSS |
| **CN (Aliexpress)** | ≤ €150 | Aliexpress è **IOSS** → IVA riscossa al checkout, no handling, no dogana fisica nel 95% dei casi |
| **Svizzera (CH)** | qualsiasi | IVA 22% + handling, come extra-UE |

## B. Handling fee corrieri 2025-2026 (orientativo)

Quando un pacco passa dogana fisica, il corriere addebita una **"commissione di sdoganamento / handling"**:

| Corriere | Fee Italia (orientativo) |
|----------|--------------------------|
| **Poste Italiane** | €7.50 fisso (sdoganamento standard) — il più basso |
| **DHL Express** | €11-18 (variabile per area + 2.5% del valore IVA, minimo) |
| **UPS** | €15-25 (advancement fee) |
| **FedEx** | €15-25 |
| **TNT** | €11-18 (oggi parte di FedEx) |
| **GLS internazionale** | €10-18 |
| **Brokerage Bring/PostNL** (small) | €5-12 |

**Trick**: per ordini USA/UK piccoli (<€80), preferire spedizioni via **postal service** (USPS → Poste Italiane, Royal Mail → Poste) invece di corriere express, perché:
- Poste IT applica €7.50 fissi vs €15-25 dei corrieri.
- Tempi più lunghi (10-20 gg vs 3-7) ma costo finale spesso **€10-20 in meno**.

## C. Tempi sdoganamento Italia 2025-2026

- **USPS via Poste IT**: 10-25 gg (variabile, Milano Linate o Roma Fiumicino).
- **Royal Mail via Poste IT**: 7-20 gg.
- **DHL Express USA → IT**: 3-7 gg (anche con dogana).
- **UPS USA → IT**: 4-8 gg.
- **FedEx**: 4-8 gg.
- **Da DE/NL/CZ/PL via corriere**: 2-5 gg, **no dogana**.
- **Da UK via corriere**: 3-7 gg + tempo dogana = 5-12 gg totali.

**Aliexpress (CN)**: **AE Premium / Choice** ora 7-15 gg con IOSS, no dogana visibile per quasi tutti gli ordini.

## E. Shop UK post-Brexit: quando vale la pena

UK aggiunge **dogana + handling** sempre. Per pochi prodotti specifici resta utile.

| Shop UK | Punto forte | Vale la pena se… |
|---------|-------------|------------------|
| **Element Games** | Citadel/Vallejo/AK -20/-30%, Pro Acryl pieno | Ordine grande >€80 → sconto compensa dogana |
| **Maple Airbrush** | Stynylrez Badger, H&S parti | Stynylrez non si trova EU, vale |
| **Hiroboy** | Mr. Hobby completo, Zero Paints | Mr. Hobby colori non bestseller |
| **Wayland Games** | Citadel scontato | Concorrente Element |
| **Goblin Gaming** | Citadel scontato | idem |
| **Triple Helix Wargames** | Mini paints + terrain | idem |

**Trucco anti-dogana UK**:
- Ordini **>€150**: paghi comunque dazio, NON spezzare in più ordini < €150 se sono pagati IVA. Spezzarli può però evitare il dazio sulla **categoria** (raro per hobby/filamenti = 0% dazio comunque).
- Element Games e Wayland **fanno IOSS** per ordini <€150 → IVA al checkout, **niente handling fee**. Verificare al checkout.
- Sempre **dichiarare valore reale**: dichiarare meno è frode doganale, rischio sequestro.

## G. Checklist anti-sorprese-dogana

1. **Prima di ordinare extra-UE**: calcola **prezzo + spedizione + IVA 22% + handling €8-25**. Se il totale è simile/superiore al prezzo EU, scegli EU.
2. **Verifica IOSS**: se il venditore mostra "IVA inclusa per Italia" al checkout, di solito è IOSS → nessuna sorpresa.
3. **Indirizzo completo + telefono**: dogana e corriere chiamano per pagamento. Indirizzi incompleti = ritardi 2-4 settimane.
4. **Codice fiscale**: per **alcuni** corrieri serve nel modulo doganale. Tienilo pronto.
5. **Conserva tracking + fattura**: per eventuali rimborsi IVA o contestazioni.
6. **Spray bombolette aerosol**: alcuni corrieri (DHL, FedEx) **rifiutano spedizioni internazionali** o richiedono dichiarazione "dangerous goods" → costi extra €30-80. **Comprare bombolette in EU sempre**, mai da USA/UK.
